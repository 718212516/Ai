Class

构造方法：

Class(constructor:Function):class

传入构造方法，返回一个类（封装好的函数方法），constructor中this指向当前类，具有以下公共方法

静态方法：

extend(subType, superType):void

混合寄生原型链模式，继承superType到subType

公共方法：

method(obj:Object):class

obj以key-value形式，将所有key对应的方法设置为class的原型方法，this指向当前类

statics:(obj:Object):class

obj以key-value形式，将所有key对应的属性设置为class的静态属性

extend(constructor:Function):class

传入子类构造函数，生成一个子类，constructor中this指向当前类

示例：

require('../module/class', function(Class) {
	var Person = Class(function() {
		this.name = 'army';
	}).methods({
		myName: function() {
			return this.name;
		}
	});
	var p = new Person;
	console.log(p.myName());
	var Man = Person.extend(function() {
		Person.call(this);
		this.sex = 'male';
	}).methods({
		mySex: function() {
			return this.sex;
		}
	});
	var m = new Man;
	console.log(m.myName() + ', ' + m.mySex());
});
